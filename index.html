Team Project

Astronomy Object Classification App

This project is a web-based data science application built to classify celestial objects—Stars, Quasars, and Galaxies—using the Sloan Digital Sky Survey (SDSS) dataset. As part of a 5-member team, we designed and implemented a complete machine learning pipeline that included: Data input & preprocessing, Model selection & evaluation, Visualization of predictions, and Exporting classification results. My primary responsibility focused on developing the data input interface, which allows users to: Upload data via CSV, and Manually input individual object data with required features (alpha, beta, redshift, u, g, r, i, z)
